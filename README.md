# 📈 Interactive Stock Market Dashboard

A real-time financial data visualization tool built with *Python* and *Streamlit*. This dashboard allows users to analyze stock performance, visualize historical trends, and examine technical indicators for any ticker available on Yahoo Finance.

🔗 *Live Demo:* [devhadkiya-stocks.streamlit.app](https://devhadkiya-stocks.streamlit.app)

---

## 🚀 Features
* *Real-time Data:* Fetches live market data using the yfinance API.
* *Interactive Visualizations:* Dynamic price charts and volume analysis powered by Plotly.
* *Custom Indicators:* Implementation of technical analysis tools (e.g., Moving Averages).
* *User Inputs:* Ability to search for any stock ticker and define custom date ranges for analysis.

## 🛠️ Tech Stack
* *Frontend:* Streamlit
* *Data Processing:* Pandas, NumPy
* *Financial Data:* YFinance
* *Visualization:* Plotly

## 📋 Requirements
To run this project locally, ensure you have the following installed:
- Python 3.8+
- Streamlit
- Pandas
- Plotly
- yfinance
